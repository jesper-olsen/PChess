CGIskak - Preferences

Here you can set the game strength. The engine iteratively searches the current position for the best move. Each iteration the search depth is extended by one ply until a stop criteria is met:

  • The search depth exceeds max_ply
  • The number of positions searched exceeds max_search
Below you can set the value of these two parameters. max_search is typically the more useful of the two - since it correlates better with time. In both cases, however, a larger value means stronger game play (and longer response time).
